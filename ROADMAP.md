# 🗺 PlayHub Project Roadmap

Цель проекта: Превратить энциклопедию игр в полноценный персональный игровой хаб с трекингом, рекомендациями и глубокой аналитикой.

---

## ✅ Реализовано (Completed)

### 🏗 Базовый функционал
- [x] Глобальный поиск по RAWG API (500k+ игр).
- [x] Детальные страницы игр (разработчики, платформы, ESRB, медиа).
- [x] Галерея скриншотов со свайпами на мобильных устройствах.
- [x] Умные фильтры с сохранением состояния в URL (Deep Linking).
- [x] Полная адаптивность (Mobile-first дизайн).

### 🚀 Phase 1 — Коллекция & Трекинг (Core)
- [x] **Статусы игр:** Добавлены `Playing` / `Completed` / `Backlog` / `Dropped` / `Wishlist`.
- [x] **Персональная оценка:** Система рейтинга (1–10 звезд).
- [x] **Заметки:** Поле для личных отзывов и мыслей об игре.
- [x] **Трекинг прогресса:** Ввод сыгранных часов и выбор платформы прохождения.
- [x] **Даты:** Автоматическая фиксация даты добавления и ручной ввод даты завершения.

### 🔍 Phase 2 — Discovery & Фильтры (Core)
- [x] **Расширенные фильтры:** Фильтрация по времени прохождения (Short/Long/Epic) и мультиселект жанров.
- [x] **Random Game:** Кнопка «Surprise Me!» с анимацией рулетки.
- [x] **Умные рекомендации 2.0:** Учет личных оценок, жанров и тегов с пояснением причин рекомендации.
- [x] **Persistence:** Сохранение активных фильтров между сессиями (localStorage).

### 📊 Phase 3 — Статистика & Профиль
- [x] **Дашборд пользователя:** Общее кол-во игр, статистика по статусам (диаграмма в планах).
- [x] **Аналитика:** Средний личный рейтинг и суммарное время прохождения.
- [x] **My Hardware:** Возможность указать свои консоли/девайсы для уточнения рекомендаций.
- [x] **Интеграция в навигацию:** Ссылка на профиль из Sidebar.

---

## 🚀 В процессе / Следующие шаги (Upcoming)

### 🛒 Phase 4 — Где купить / поиграть (Completed)
- [x] **Магазины:** Ссылки на официальные площадки (Steam, PS Store, Xbox, Nintendo).
- [x] **CheapShark Integration:** Сравнение цен и отображение текущих скидок.
- [x] **Game Pass / Plus:** Метки «Доступно в подписке» (по тегам).
- [x] **Retro Integration:** Ссылки на Internet Archive для старых игр (запуск в браузере).

### ⚡️ Phase 5 — UX & Производительность (Medium Priority)
- [ ] **Offline Mode:** Service Worker + кэширование RAWG-ответов.
- [x] **Skeletons:** Замена спиннеров на скелетную загрузку (GameCardSkeletons реализованы).
- [ ] **Side-by-side:** Сравнение двух игр на одном экране.
- [x] **Public Profiles:** Возможность поделиться ссылкой на свою коллекцию (реализовано через /collection/:uid).

### 🔌 Phase 6 — Дополнительные интеграции (Low Priority)
- [ ] **Library Import:** Импорт игр из внешних сервисов (MVP — Steam).
  - [ ] Steam (по SteamID через публичные профили).
  - [ ] PlayStation / Xbox (исследование неофициальных API и OAuth).
  - [ ] Импорт из файлов (экспорт/импорт CSV или JSON).
- [ ] **IGDB API:** Обогащение данных (движки игр, игровые режимы).
- [ ] **YouTube Data API:** Автоматический поиск OST и видеообзоров прямо на странице игры.
- [ ] **GNews API:** Лента актуальных новостей по конкретной игре или франшизе.
